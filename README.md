# 🎬 Netflix Content Strategy Analysis

> **Live Portfolio:** [minga-portfolio.onrender.com](https://minga-portfolio.onrender.com)

Exploratory data analysis of Netflix's content catalog covering 7,787 titles 
across movies and TV shows from 2008 to 2021.

---

## 📊 Key Findings

- 🎬 Netflix catalog is **69.1% Movies** (5,377) and **30.9% TV Shows** (2,410)
- 📅 Peak content growth was **2019** — 2,153 titles added in a single year
- 🇺🇸 **United States** is the top content producing country by far
- 🌍 **International Movies** is the most common genre on the platform
- 🎥 Average movie duration is **99 minutes**
- 📺 Average TV show runs **1.8 seasons**
